Crisi abitativa

Prezzi, affitti, redditi e accesso alla casa

Dashboard costruita per leggere la crisi abitativa su due scale: confronto europeo sugli indicatori armonizzati e focus locale italiano su prezzi, affitti e redditi comunali quando i dati territoriali sono disponibili.

Fonte: Eurostat, OMI, MEF Grafico europeo via API Eurostat. Focus locale predisposto per export dal repository. Apri repository Crisi_abitativa
1

Scegli l'indicatore

Nel confronto europeo il menu cambia la variabile; l'Italia resta sempre evidenziata e il range mostra minimo e massimo tra paesi.

2

Confronta pochi paesi

Usa il filtro paesi per aggiungere linee selezionate. Troppe linee rendono il grafico meno leggibile.

3

Passa al locale

Nel focus Italia scegli regione e misura OMI. I valori comunali sono proxy territoriali e vanno letti con le note metodologiche.

Note metodologiche

  • Il confronto europeo usa indicatori armonizzati Eurostat. La disponibilita' temporale puo' cambiare tra paesi e indicatori.
  • Il range min-max e' calcolato sui paesi disponibili nell'anno, escludendo gli aggregati come EU27.
  • Nel focus locale prezzi e affitti OMI sono quotazioni territoriali per zona, non prezzi effettivi di transazione o canoni di contratto.
  • Medie e mediane comunali OMI non sono pesate per transazioni, stock abitativo, popolazione o superficie.

Fonte e repository

Il codice sorgente del progetto dati e' nel repository NazarenoLecis/Crisi_abitativa.

La dashboard del sito consuma dati gia' puliti o endpoint pubblici. Le API e gli export pesanti non vengono eseguiti direttamente nel browser.

Per aggiornare il focus locale bisogna esportare dal repository Python i JSON regionali in data/crisi-abitativa/regions/ e aggiornare local_index.json.
