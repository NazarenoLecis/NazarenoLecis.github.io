Pensioni italiane

Pensioni: trattamenti, pensionati, spesa e contributi

Dashboard statica alimentata da INPS Open Data, Appendice statistica al XXV Rapporto annuale, Casellario dei pensionati e serie storiche delle aliquote IVS. Le pensioni sono trattamenti; i pensionati sono persone: le due misure restano sempre separate.

Payload dati da Cloudflare R2 Repo dati: pensioni_italia. Pubblicazione: nazarenolecis-data-pipeline. Mappa regionale ora; dettaglio provinciale da integrare quando la query API INPS sara' stabilizzata. Apri repository dati
