Metodo

Cosa si puo leggere ora

Il sito separa visualizzazione, pubblicazione dati e pipeline di estrazione. Ogni limite visibile qui dipende dai JSON pubblicati, non da logica nascosta nella dashboard.

Dati disponibili

Conteggi dei reati denunciati da fonte ISTAT, normalizzati in formato lungo e pubblicati come JSON. La granularita territoriale include Italia, regioni, province e codici capoluogo quando presenti nel dataset.

Dati da integrare

Autori, vittime, cittadinanza, sesso, eta e separazione piena tra reati violenti e altri reati richiedono basi dati o dizionari aggiuntivi. La dashboard evidenzia questi campi come non popolati finche il pipeline non li produce.

Architettura

Il repo sicurezza genera JSON, il repo data pipeline pubblica su Cloudflare R2, questo repo GitHub Pages legge i dati pubblici e costruisce l'interfaccia.
