Bilancio pubblico italiano

Dashboard che racconta entrate e spesa pubblica con indicatori principali dal 2002 al 2025. Dati da MEF, Eurostat e OCSE, normalizzati in JSON nel repo Bilancio_pubblico.

Fonte: MEF, Eurostat, OCSE Il JSON viene rigenerato da una pipeline separata, questo repo resta solo frontend. Elaborazione di Nazareno Lecis.
1

Input stabile

Un endpoint JSON contiene tutti i blocchi richiesti dalla dashboard (kpi, serie e tabelle).

2

Interattivo

Puoi confrontare Italia, UE e area euro con la metrica che preferisci.

3

Trasparenza

Fonte, periodo e data di aggiornamento sono visibili direttamente nel blocco metodo.
